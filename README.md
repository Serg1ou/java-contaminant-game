# Java Contaminants Game

Academic Java project simulating a grid-based environment with multiple autonomous robots.

## Description
This project was developed as part of a university programming assignment.
It focuses on object-oriented design, deterministic movement logic.

## Features
- Cleaning robot controlled by keyboard input
- Two contaminant robots with different movement strategies
- Coin placement and collection mechanics
- Win condition handling

## Goal
The goal was to implement correct logic under predefined rules and tests.

## Notes 
Originally developed as a university project (2023). Refactored and published to GitHub in 2026.
